# 💻 Desenvolvimento ABAP Backend - SAP S/4HANA

> <sub>**Hub Central de Lógica e Processamento SAP**</sub>
> <sub>Este repositório serve como base de conhecimento, laboratório de testes e versionamento de soluções exclusivas para a **camada de Backend** do ERP SAP S/4HANA. Focado na centralização de scripts, lógicas de negócio, extração de dados e integrações do desenvolvimento ABAP clássico e moderno.</sub>

---

## 🚀 Stack Tecnológica

> <sub>*O repositório aborda soluções envolvendo as seguintes ferramentas e tecnologias:*</sub>

### ⚙️ Core Development (S/4HANA)
* **ABAP Clássico & Moderno:** Lógicas de processamento, extrações (Open SQL) e ABAP OO (Orientação a Objetos).
* **Extensões (Enhancements):** BAdIs, User Exits e adaptações do standard SAP.
* **Integração:** Criação de serviços (SEGW) para consumo via APIs REST/OData.

### 🛠️ Ferramentas & IDEs
* **VS Code:** Edição ágil, formatação com extensões ABAP e integração com Git.
* **SAP GUI / Transações Core:** SE11, SE24, SE37, SE38 e SE80.

---

## 📂 Estrutura do Repositório

> <sub>*Os projetos estão organizados por componentes técnicos da arquitetura ABAP:*</sub>

| Diretório | Descrição |
| :--- | :--- |
| **📚 Fundamentos_e_Sintaxe** | Exercícios de lógica, loops, strings e variáveis. |
| **📊 Relatorios_ALV** | Extração via Open SQL e exibição em grades interativas. |
| **🗄️ Dicionario_de_Dados** | Criação de tabelas (Z), estruturas, elementos de dados (SE11). |
| **🧩 Orientacao_a_Objetos** | Classes, interfaces, heranças e métodos modernos (SE24). |
| **🛠️ Modulos_Funcao_e_BAPIs** | Funções reutilizáveis (SE37) e consumo de integrações padrão. |
| **🌐 Integracoes_e_OData** | Criação de serviços Backend for Frontend (BFF) via SEGW. |
| **⚡ Enhancements_e_BAdIs** | Ampliações e alterações de comportamento do padrão SAP. |
